Syntax:

  • F: move forward by line length and draw line
  • f: move forward by line length without drawing a line
  • +: turn left by angle
  • -: turn right by angle
  • [: push current position and rotation onto the stack
  • ]: pop current position and rotation off the stack
  • Non-terminals can be any character in production rules

Starting axiom:

F+F+F+F

Input rules:

F => F+FF++F+F

Iterations:

Angle:

90

Line length:
